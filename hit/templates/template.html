Welcome! Thank you for participating in this task. Please read the instructions carefully.

You will see a snippet of a converation between a human user and a chatbot below. The chatbot, named Chattie, is supposed to help the user with their calendar. Unfortunately, Chattie is not sure it understood the last instruction that the user gave (in italics). Luckily, Chattie can offer some options for what it thinks the user meant.

Your task is to evaluate whether any of the options given by Chattie matches the original instruction. If one of the options is correct, please select it and press submit.

You can click the buttons or use the number keys (1,2,3,4) to select options. Note that you may have to click the HIT once to get the numbers to work, and clicking on the text box will disable the numbers (options can still be clicked with the mouse).

If none of the options proposed by Chattie match what you think the user wanted, then please use the textbox to rephrase what you think the user meant. Try to rephrase it in a way you think Chattie would understand.
